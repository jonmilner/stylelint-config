## Usage

Install packages as devDependencies:

```bash
npm install --save-dev stylelint@7.1.0 http://github.com/jonmilner/stylelint-config
```

Add a `.stylelintrc` config to your project:

```json
{
  "extends": "stylelint-config"
}
```
